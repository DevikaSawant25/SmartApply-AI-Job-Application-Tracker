# SmartApply: AI-Powered Job Application Tracker
SmartApply is a self-hosted, full-stack application designed to help you track your job applications and optimize your resume using AI. It provides a visual Kanban board to manage your application pipeline, compares your resume to job descriptions to find skill gaps, and generates tailored cover letters.
---
## 🏗️ Architecture Overview
The project is structured as a decoupled full-stack application:
```text
smartapply/
├── backend/             # FastAPI Backend Service
│   ├── database.py      # Database connection pool using asyncpg
│   ├── models.py        # Pydantic schemas for data validation
│   ├── ai.py            # Anthropic Claude API prompt engineering
│   └── main.py          # REST API router and application lifecycle
├── frontend/            # Streamlit Frontend Client
│   ├── api.py           # HTTP client using httpx to communicate with backend
│   └── app.py           # Interactive UI (Kanban, forms, analytics)
├── database/            # SQL Schemas
│   └── schema.sql       # PostgreSQL table schemas
├── .env                 # Local secrets (API keys, database URLs) - DO NOT COMMIT
├── .gitignore           # File exclusion rules for Git
├── requirements.txt     # Python dependencies list
└── README.md            # You are here!
```

## 🛠️ Step-by-Step Setup Guide
Since you are building and learning this on your own, here are the step-by-step commands to set up the project:
### 1. Set Up Your Virtual Environment
To keep your Python packages isolated and prevent system conflicts, create a virtual environment in the project directory:
```bash
# Create the virtual environment folder named 'venv'
python -m venv venv
# Activate the virtual environment
# On macOS/Linux:
source venv/bin/activate
# On Windows:
# venv\Scripts\activate
```
### 2. Install Project Dependencies
With the virtual environment activated, install the required packages:
```bash
pip install fastapi uvicorn asyncpg pydantic streamlit httpx anthropic python-dotenv pandas
```
To save this snapshot of packages:
```bash
pip freeze > requirements.txt
```
### 3. Database Setup (PostgreSQL)
Ensure you have PostgreSQL installed and running on your machine.
1. **Log in to PostgreSQL CLI:**
   ```bash
   psql -U postgres
   ```
2. **Create the Database:**
   ```sql
   CREATE DATABASE smartapply;
   ```
3. **Exit PostgreSQL CLI:**
   ```sql
   \q
   ```
4. **Apply the Schema:**
   Run the schema SQL script to set up the `jobs` table:
   ```bash
   psql -U postgres -d smartapply -f database/schema.sql
   ```
### 4. Configuration (`.env`)
Create a `.env` file in the root of the project and add your credentials:
```ini
DATABASE_URL=postgresql://postgres:yourpassword@localhost:5432/smartapply
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

## 🚀 Running the Application
To run the full stack, you need to start the backend API server and the Streamlit frontend client in two separate terminal windows.
### **Terminal 1: Start the FastAPI Backend**
```bash
source venv/bin/activate
uvicorn backend.main:app --reload
```
* The backend will run at: **`http://localhost:8000`**
* You can access the auto-generated API docs at: **`http://localhost:8000/docs`**
### **Terminal 2: Start the Streamlit Frontend**
```bash
source venv/bin/activate
streamlit run frontend/app.py
```
* The frontend will run and automatically open in your browser at: **`http://localhost:8501`**
